Picking up a new language and some discipline

 Sun 31 March 2013 by James Polera

Picking up a new language is exciting, and there are so many great new ones out there. On my white board (which is really orange, but I digress) I have 6 languages to learn for 2013:

  • Clojure
  • Go
  • Scala
  • F#
  • Erlang
  • Elixir

It seems like every year I have these swings in language desires. Spring seems to swing towards Clojure (oh, the JVM puns!). Summer runs Erlang. Fall/Winter falls squarely in the Scala camp. I want to learn all of these languages, but I feel like I may set unrealistic goals for myself given the amount of time I have to dedicate to learning them.

That changes this year, this month with this blog post. I'm going to apply the Seinfeld Calendar method to track/prod my progress. I'm going to stick to one language until I'm comfortable with it (meaning that I can build something new with it or port an old project to it), and I intend to post something I find interesting about the language I'm currently undertaking here.

I realize, of course, that I may not be able to meet this goal for all 6 languages this year, but that's fine. Full disclosure - I do have varying levels of exposure to all of these languages from tinkering over the last year or so. It's time to build now.

Set some goals, learn some things in the process. All positive.
